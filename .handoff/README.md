# .handoff

This directory stores ChatGPT/OpenCode continuity files for this project.

- `latest.md`: concise current handoff
- `report.md`: latest deep report
- `status.md`: current status snapshot
- `archive/`: timestamped older handoffs and reports

Do not store secrets here.

Legacy note: older handoff files may still exist in `.ai/`. New handoffs should use `.handoff/`.
